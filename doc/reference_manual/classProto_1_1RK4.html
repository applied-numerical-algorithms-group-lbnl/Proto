Detailed Description

template<class X, class F, class dX>
class Proto::RK4< X, F, dX >

Generic Explicit RK4 Algorithm.

class X is the type of your state data. class F provides an evaluation of the X' for a specified state of X and the time. If your X' is determined entirely from the current state values then F can ignore. F must implement the function:

void operator(X& k, double a_time, double a_dt, const X& a_state, const X& a_shift);

F interprets the real state as a_state+a_shift
